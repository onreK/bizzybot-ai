## 3. Class & filing basis
- **International Class:** **042** (Software as a Service / non-downloadable software)
- **Filing basis:** **Section 1(a) — use in commerce** (the product is live and has a
  paying customer, so you file as already in use, not "intent to use").

## 5. Dates of first use — **[CONFIRM — you must supply these]**

service under the name "BizzyBot" to a customer.
- **Date of first use ANYWHERE:** **[CONFIRM: earliest date you used "BizzyBot" publicly —

- **Date of first use IN COMMERCE:** **[CONFIRM: first date a customer could actually sign

## 6. Specimen (proof of use — required for a 1(a) filing)
USPTO needs proof the mark is used for the service. Provide a **screenshot of the live
site** that clearly shows BOTH:
1. the **"BizzyBot" name/logo**, AND
2. the **service being offered** (the AI receptionist description + a way to sign up/buy).

- **Best specimen:** a screenshot of **bizzybotai.com** homepage (shows the BizzyBot brand
  + "AI receptionist" service + Start/Pricing CTA) — capture the full page including the
  visible URL. A second option: the **/pricing** page (shows it's an actual paid offering).
- Save as PNG/JPG. Note the date captured.

## 7. Optional second filing — the logo (design mark)
- File the **BizzyBot logo** as a separate "design mark" if you want to protect the visual

## 8. Cost & path summary
| Path | Government fee | Service/attorney fee | Notes |
|---|---|---|---|
| DIY (uspto.gov Trademark Center) | ~$350 / class | $0 | Cheapest; you handle any refusals |
| Flat-fee service (LegalZoom, Trademark Engine) | ~$350 / class | ~$200–600 | Middle ground; confirm it includes a clearance search |
| Trademark attorney | ~$350 / class | ~$500–1,500 | **Recommended** — proper clearance search + handles Office Actions |

*(Government fees changed in 2025 — verify the current Class-042 base fee at uspto.gov before filing.)*

## 9. Before you file — the one thing that matters most
Have the attorney / service run a **clearance search** first. The prior "BIZZYBOT" app
(serial 98377405) is **dead/abandoned** (verified 2026-07-22 on USPTO TSDR), so the big
blocker is gone — but the name is crowded (e.g., **bizzybotnj.com**, a solo AI-automation
*services* agency in NJ, may hold limited local common-law rights in a different lane).
The clearance search confirms you're truly clear to register. This is the step worth
paying for.

## 10. Right now (free, no filing needed)
Start putting **™** after "BizzyBot" on the site/marketing — it signals you're claiming the
mark and requires no registration. (The **®** symbol may only be used AFTER registration.)
